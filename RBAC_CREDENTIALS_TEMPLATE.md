# RBAC Credentials Template (Do Not Store Real Passwords In Repo)

This file is a template only. Do not put real passwords here.

## Recommended approach
- Store real credentials in a password manager
- Use Supabase Auth "Users" to manage test accounts
- Rotate any credentials that were previously shared in logs or public assets

- REGISTERED
- INVESTOR
- VENDOR

- ADMIN
- SUPER_ADMIN
